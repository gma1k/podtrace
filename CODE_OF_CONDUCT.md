# Code of Conduct

The podtrace project adopts the **Contributor Covenant, version 2.1**
as its code of conduct. The full text is available at:

<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

In short:

- We pledge to make participation in podtrace a respectful and
  welcoming experience for everyone, regardless of background or
  identity.
- Constructive feedback, helpfulness, and empathy are expected.
- Disrespectful, abusive, or discriminatory behavior is not
  tolerated and will result in moderation action ranging from a
  warning up to a permanent ban from project spaces.

This Code of Conduct applies in all project-managed spaces — issue
tracker, pull requests, discussions, and any official off-platform
communication — and when an individual is officially representing
podtrace in public spaces.

## Reporting

To report a concern, contact the maintainer privately at:

**ghassan+podtrace@malke.nl**

All reports are reviewed and investigated promptly and fairly. The
maintainer is obligated to respect the privacy and security of the
reporter of any incident.

## Enforcement

The maintainer applies the [Contributor Covenant Enforcement
Guidelines](https://www.contributor-covenant.org/version/2/1/code_of_conduct/#enforcement-guidelines)
when determining the consequences for any action deemed in violation
of this Code of Conduct. The four enforcement levels are correction,
warning, temporary ban, and permanent ban.
